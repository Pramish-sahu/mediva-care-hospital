# Internal Pages — Prompt 5

The Prompt 5 package replaces all internal-page placeholders with complete, responsive healthcare content.

## Completed pages

- About
- Departments
- Doctors
- Doctor details
- Health packages
- Testimonials
- Gallery
- Contact
- Privacy policy
- Terms of use
- 404

## Shared files

- `assets/css/internal.css` provides page heroes, breadcrumbs, grids, cards, forms, filters, legal layouts, gallery lightbox, and responsive rules.
- `assets/js/internal-pages.js` provides searchable filters, gallery dialog behavior, newsletter feedback, and legal section navigation.
