- [ ] Run `python3 scripts/validate-project.py`.
- [ ] Test mobile, tablet, laptop, and desktop layouts.
- [ ] Test light, dark, reduced-motion, and print modes.
- [ ] Run Lighthouse against the deployed URL.
- [ ] Review browser console and network errors.
- [ ] Confirm no horizontal overflow.

## Deployment

- [ ] Update `sitemap.xml` and `robots.txt` for the final domain.
- [ ] Configure the custom domain and HTTPS.
- [ ] Configure security and cache headers.
- [ ] Confirm the 404 page works on the selected host.
- [ ] Create and archive a final release ZIP.
